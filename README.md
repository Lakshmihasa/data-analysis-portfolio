# Data Analysis Portfolio

This repository contains my data analytics projects demonstrating skills in data analysis, machine learning, and visualization using Python.

## Projects

### 1. Sales Analysis
Analysis of sales data to identify revenue trends and business insights.

### 2. Healthcare Readmission Analysis
Machine learning model to predict hospital readmissions.

### 3. Sports Analytics
Performance analysis of players and teams using sports data.

### 4. Financial Market Analysis
Stock trend analysis, risk assessment, and portfolio optimization.

### 5. E-commerce Analytics
Customer segmentation using RFM analysis and recommendation system using Apriori algorithm.

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Mlxtend
